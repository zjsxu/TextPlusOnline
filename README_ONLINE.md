# 🌐 TextDiff+ 在线版本

## 🎯 项目简介

TextDiff+ 是一个专业的文本对比分析工具，现已支持在线使用！无需安装任何软件，直接在浏览器中即可使用。

### ✨ 主要功能

- 📄 **多格式支持**: PDF、TXT、DOCX 文件智能解析
- 🔍 **精准对比**: 基于 diff-match-patch 的高精度文本对比
- 🧠 **语义分析**: 功能词识别和语气强度分析
- 📊 **可视化图表**: 实时生成分析图表
- 📈 **词频统计**: 中英文词频分析和对比
- 📚 **词典管理**: 自定义功能词词典
- 🎨 **主题系统**: 多种主题和自定义颜色
- 🏷️ **智能标注**: Mini-Doccano 风格的文本标注

### 📱 使用步骤

1. **打开网站** - 在浏览器中访问在线版本
2. **上传文件** - 支持拖拽上传 PDF、TXT、DOCX 文件
3. **粘贴文本** - 也可以直接粘贴文本内容
4. **开始对比** - 点击"执行对比"按钮
5. **查看结果** - 查看详细的对比结果和分析图表

### 🎨 界面特色

- **中英双语界面** - 完整的中英文对照
- **右侧图表面板** - 实时显示语义分析图表
- **响应式设计** - 适配桌面和移动设备
- **主题切换** - 默认、高对比度、自定义主题

### 🔧 技术特点

- **纯前端实现** - 无需服务器，数据不上传
- **隐私保护** - 所有处理都在本地浏览器中完成
- **快速响应** - 基于现代Web技术，响应迅速
- **跨平台兼容** - 支持所有现代浏览器

### 📊 适用场景

- **学术研究** - 论文版本对比和分析
- **文档管理** - 合同、报告等文档的变更追踪
- **内容编辑** - 文章修改前后的对比分析
- **语言学研究** - 文本语气和语义分析
- **翻译对比** - 不同版本翻译的对比
